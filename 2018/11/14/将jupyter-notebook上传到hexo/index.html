将jupyter notebook上传到hexo

发表于 2018-11-14

通过安装插件实现，按照官网的步骤如下：

第一步：

在hexo的安装目录运行npm install hexo-jupyter-notebook --save

我使用的是nvm进行管理的 需要先进行npm的激活采用 nvm use v版本号

第二步：

安装pandoc: sudo apt-get install pandoc

安装nbconvert: pip install nbconvert

第三步：

  • 打开hexo的配置文件 _config.yml， 把里面的post_asset_folder: true改为true

  • 在_post目录下新建专门用于存放.ipynb文件的目录:mkdir jupyter-demo

  • 将需要展示的文件存放到jupyter-demo目录下

  • 在需要展示的地方写入

以上就可以在hexo博客中显示jupyter notebook里的内容。
